# How-To

1. Install terraform following: https://developer.hashicorp.com/terraform/tutorials/aws-get-started/install-cli
2. Navigate to terraform folder:
   ```
   cd terraform
   ```
4. Initialize terraform:
   ```
   terraform init
   ```
6. Check changes and apply them:
    ```
    terraform plan
    terraform init
    ```
